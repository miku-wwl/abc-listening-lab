## 3. 新问题 vs 旧问题

### 重复出现的问题

1. 技术专名、缩写和产品/方法名容易被 ASR 或大脑吸附为普通词：`Force.com`、`time sharing`、`utility computing`、`provisioning`、`capital/operational expense` 和 subject code 易错。
2. 长句中对象、动作、条件和结果混在一起，容易只记住最后一个术语。
3. 屏幕演示中的 `this / here / that one / the first one` 缺少画面后很难恢复指代。
4. 数字、节点、版本、概率或课程规则需要和所属对象绑定，不能只记数字本身。
5. 本次仍无裸听 notes，不能把“材料中的高风险点”写成“你本人听错的点”。
